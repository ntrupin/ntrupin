Welcome to my website! Here serves to showcase my writings, projects, and other odds and ends I find interesting.

About this site

This site is written in Python using the Flask web framework with Jinja2 for templating, PostgreSQL as its database, and Werkzeug for authentication. The frontend uses vanilla HTML/CSS/JavaScript with Pico.css for base styling. I currently host the database on Heroku and the client/server on Vercel. GitHub CI/CD manages deployment from this GitHub repository.

Started as a static site hosted on GitHub Pages in 2017, ntrupin.com now serves as a dynamic hub for my projects, writing, and more.
